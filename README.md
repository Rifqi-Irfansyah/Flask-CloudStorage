Installation
1. Clone Repository
git clone https://github.com/Rifqi-Irfansyah/Flask-CloudStorage
cd flask-gallery

2. Create Virtual Environment
python -m venv venv

Activate environment:

Windows
venv\Scripts\activate

Mac / Linux
source venv/bin/activate

3. Install Dependencies
pip install -r requirements.txt
▶️ Run Application
python app.py
